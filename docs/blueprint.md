# **App Name**: ElderEase

## Core Features:

- Easy Sign-Up/Login: Simplified sign-up and login using mobile number or email with large, easy-to-read fields and clear instructions.
- Health Support Dashboard: A dashboard presenting critical health information and options with intuitive icons and large text for quick access.
- Medicine Reminders: Automatic medication reminders with snooze/acknowledge functionality, displayed prominently with simple controls.
- Emergency SOS: One-tap emergency SOS button to immediately notify pre-selected contacts with the user's live GPS location.
- Ordering Portal: An interface to facilitate the ordering of medicine, and optionally groceries, using clear visuals, and minimizing steps.
- AI Diet Planner: A diet planner that offers personalized, relevant meal suggestions based on user inputs regarding age, conditions, and preferences. An LLM tool is used to evaluate the appropriateness of various foods and suggest complete meals.
- GPS Tracker: Safe location tracking allows authorized family members to monitor the user's real-time location via GPS, with appropriate privacy settings.

## Style Guidelines:

- Primary color: Soft blue (#89B3E5) to evoke a sense of calmness and reliability.
- Background color: Very light blue (#F0F4F9), nearly white, creating a clean and unobtrusive backdrop.
- Accent color: Gentle green (#A0D6B4) for positive actions and confirmations, such as acknowledging a reminder.
- Body and headline font: 'PT Sans' (sans-serif) for readability, accessibility, and a modern but friendly feel.
- Use simple, universally recognized icons with sufficient contrast to the background.
- Maintain a clear, linear layout with generous spacing and minimal visual clutter.
- Subtle animations to confirm actions and guide the user through the interface.
